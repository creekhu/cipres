To follow along with the quickstart tutorial:

- Put credentials in demo.rc and source it.

- Download the sample file.

- Run "sh submit.sh"

- If the account you used has a valid email address, you'll get mail on job completion.

- Poll with "sh get.sh job_url"

- List results with "sh get.sh job_url/output" when job is finished

- Download STDOUT (or other file) by finding downloadUri of the file in the results and running
"sh download.sh downloadUri.url"

- Delete job with "sh delete.sh job_url"

- List jobs with "sh get.sh $URL/job/$CRA_USER"
