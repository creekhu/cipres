UNROOT

UNROOT unroots a rooted tree using the Phylip tree editor RETREE. Retree reads in a tree, or allows the user to construct one, and displays this tree on the screen. The user then can specify how the tree is to be rearraranged, rerooted or written out to a file.

Manual: http://evolution.genetics.washington.edu/phylip/doc/retree.html
