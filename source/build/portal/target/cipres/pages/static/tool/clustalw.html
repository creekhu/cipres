CLUSTALW

CLUSTALW is a fully automatic program for global multiple alignment of DNA and protein sequences. The alignment is progressive and considers the sequence redundancy. Trees can also be calculated from multiple alignments. The program has some adjustable parameters with reasonable defaults.
INPUT = dna sequence files; protein sequence files; profiles

Manual: http://www.ebi.ac.uk/2can/tutorials/nucleotide/clustalw1.html
